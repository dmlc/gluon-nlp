Get Started
===========
This is the quick start guide for using Gluon NLP Toolkit.

Installation
------------

.. note::

    This package is under constant development, so be sure to upgrade often.

1. Install nightly build of mxnet using `pip install -U --pre $MXNET_VARIANT` where `MXNET_VARIANT` can
   be any of the listed ones at https://github.com/apache/incubator-mxnet/issues/8671.
2. Install Gluon NLP Toolkit using pip

.. code:: bash

    pip install -U --pre http://gluon-nlp-dist.s3-accelerate.dualstack.amazonaws.com/gluonnlp-0.1.0.tar.gz
